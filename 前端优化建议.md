# 前端优化建议 - 智能专利辅助审核系统

## 📊 现状分析

### 当前问题

1. **巨型组件问题** ⚠️
   - `SimplePatentChat.vue`: **3111行** - 严重超标（建议单个组件 < 300行）
   - 单个组件包含了所有业务逻辑、状态管理、UI渲染
   - 难以维护、测试、复用

2. **代码结构问题**
   - 缺少组件目录（`components/` 为空）
   - 所有逻辑耦合在一个文件中
   - 没有自定义 composables/hooks
   - 重复代码多（如文件上传、消息处理）

3. **性能问题**
   - 大量计算属性和watch可能导致性能问题
   - 没有虚拟滚动（聊天消息多时性能下降）
   - 未使用 Vue 3 的性能优化特性

4. **状态管理问题**
   - 本地状态过多（30+ ref/reactive）
   - 与 Pinia store 混用，界限不清
   - localStorage 直接操作散落各处

**useChatSession.ts** (会话管理 composable)
```typescript
import { ref, computed } from 'vue';
import api from '@/services/api';
import type { ChatSession, ChatMessage } from '@/types';

export function useChatSession() {
  const sessions = ref<ChatSession[]>([]);
  const currentSessionId = ref<number | null>(null);
  const messages = ref<ChatMessage[]>([]);
  const loading = ref(false);

  const currentSession = computed(() => 
    sessions.value.find(s => s.id === currentSessionId.value)
  );

  const loadSessions = async () => {
    try {
      loading.value = true;
      const data = await api.get('/ai/sessions');
      sessions.value = data.map(transformSession);
    } catch (error) {
      console.error('加载会话失败:', error);
      throw error;
    } finally {
      loading.value = false;
    }
  };

  const switchSession = async (sessionId: number) => {
    currentSessionId.value = sessionId;
    const session = sessions.value.find(s => s.id === sessionId);
    
    if (!session?.messages?.length) {
      const data = await api.get(`/ai/sessions/${sessionId}`);
      messages.value = data.messages;
    } else {
      messages.value = session.messages;
    }
  };

  const createSession = () => {
    currentSessionId.value = null;
    messages.value = [];
  };

  const deleteSession = async (sessionId: number) => {
    await api.delete(`/ai/sessions/${sessionId}`);
    sessions.value = sessions.value.filter(s => s.id !== sessionId);
    
    if (currentSessionId.value === sessionId) {
      createSession();
    }
  };

  return {
    sessions,
    currentSessionId,
    currentSession,
    messages,
    loading,
    loadSessions,
    switchSession,
    createSession,
    deleteSession,
  };
}
```

**useChat.ts** (聊天逻辑 composable)
```typescript
import { ref } from 'vue';
import type { ChatMessage } from '@/types';
import { useStreaming } from './useStreaming';

export function useChat() {
  const messages = ref<ChatMessage[]>([]);
  const inputMessage = ref('');
  const isLoading = ref(false);
  const currentResponse = ref('');

  const { startStream, stopStream } = useStreaming();

  const sendMessage = async (
    content: string, 
    attachments?: any[], 
    options?: any
  ) => {
    if (!content.trim() && !attachments?.length) return;

    // 添加用户消息
    messages.value.push({
      role: 'user',
      content,
      timestamp: new Date(),
      attachments,
    });

    // 清空输入
    inputMessage.value = '';
    isLoading.value = true;
    currentResponse.value = '';

    try {
      // 发送流式请求
      await startStream({
        messages: buildMessageHistory(messages.value, content, attachments),
        onChunk: (chunk: string) => {
          currentResponse.value += chunk;
        },
        onComplete: (finalContent: string) => {
          messages.value.push({
            role: 'assistant',
            content: finalContent,
            timestamp: new Date(),
          });
        },
      });
    } catch (error) {
      console.error('发送消息失败:', error);
      throw error;
    } finally {
      isLoading.value = false;
      currentResponse.value = '';
    }
  };

  const regenerateResponse = async (messageIndex: number) => {
    const userMsg = messages.value[messageIndex - 1];
    if (!userMsg) return;

    // 删除AI回复
    messages.value.splice(messageIndex);
    
    // 重新生成
    await sendMessage(
      userMsg.content,
      userMsg.attachments,
      { templateId: userMsg.templateId }
    );
  };

  return {
    messages,
    inputMessage,
    isLoading,
    currentResponse,
    sendMessage,
    regenerateResponse,
  };
}
```

### 三、状态管理优化

#### 3.1 Pinia Store 重构

```typescript
// stores/chat.ts
import { defineStore } from 'pinia';
import { ref, computed } from 'vue';
import type { ChatMessage, ChatSession } from '@/types';

export const useChatStore = defineStore('chat', () => {
  // State
  const sessions = ref<ChatSession[]>([]);
  const currentSessionId = ref<number | null>(null);
  const messages = ref<ChatMessage[]>([]);
  const uploadedFiles = ref<any[]>([]);
  
  // Getters
  const currentSession = computed(() => 
    sessions.value.find(s => s.id === currentSessionId.value)
  );
  
  const hasAttachments = computed(() => uploadedFiles.value.length > 0);
  
  // Actions
  const addMessage = (message: ChatMessage) => {
    messages.value.push(message);
  };
  
  const clearMessages = () => {
    messages.value = [];
  };
  
  // 持久化
  const saveToStorage = () => {
    localStorage.setItem('chat_sessions', JSON.stringify(sessions.value));
    localStorage.setItem('current_session_id', String(currentSessionId.value));
  };
  
  const loadFromStorage = () => {
    const saved = localStorage.getItem('chat_sessions');
    if (saved) sessions.value = JSON.parse(saved);
    
    const savedId = localStorage.getItem('current_session_id');
    if (savedId) currentSessionId.value = Number(savedId);
  };
  
  return {
    sessions,
    currentSessionId,
    messages,
    uploadedFiles,
    currentSession,
    hasAttachments,
    addMessage,
    clearMessages,
    saveToStorage,
    loadFromStorage,
  };
});
```

#### 3.2 本地存储封装

```typescript
// composables/useLocalStorage.ts
export function useLocalStorage<T>(key: string, defaultValue: T) {
  const value = ref<T>(defaultValue);
  
  // 从 localStorage 加载
  const load = () => {
    try {
      const saved = localStorage.getItem(key);
      if (saved) value.value = JSON.parse(saved);
    } catch (error) {
      console.error(`Failed to load ${key}:`, error);
    }
  };
  
  // 保存到 localStorage
  const save = () => {
    try {
      localStorage.setItem(key, JSON.stringify(value.value));
    } catch (error) {
      console.error(`Failed to save ${key}:`, error);
    }
  };
  
  // 监听变化自动保存
  watch(value, save, { deep: true });
  
  // 初始化加载
  onMounted(load);
  
  return { value, load, save };
}
```

  error.value = err;

const reset = () => {
  hasError.value = false;
  error.value = null;
};
